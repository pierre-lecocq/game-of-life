# John Conway's Game of Life

```
usage: main.py [-h] [-f fps] [-c columns] [-r rows] [-s size] [-g gap]

John Conway's Game of Life

options:
  -h, --help            show this help message and exit
  -f fps, --fps fps     number of frames per second
  -c columns, --columns columns
                        number of columns
  -r rows, --rows rows  number of rows
  -s size, --size size  size of a cell in pixels
  -g gap, --gap gap     gap between cells in pixels
```
